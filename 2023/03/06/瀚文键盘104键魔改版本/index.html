本篇为魔改稚辉君瀚文键盘的笔记

首先是魔改的前置技能学习

本键盘的核心部件为 74hc595移位寄存器，首先要了解移位寄存器的工作原理

普通矩阵键盘的原理为：将每行每列的按键串联起来，再通过连接单片机io口，当某对io引脚电平发生变化时，通过滤波算法，可以判断哪个键被按下

=================================

= = = = = = 1.1

=================================

= = = = = = 1.2

=================================

= = = = = = 1.3

=================================

2.1 2.2 2.3 2.4 2.5 2.6

例如：当第三行第三列按键被按下时，引脚1.3与2.3会同时发生改变，从而通过扫描引脚电平变化判断按键按下情况

移位寄存器

百度百科：在数字电路中，移位寄存器是一种在若干相同时间脉冲下工作的以触发器为基础的器件，数据以并行或串行的方式输入到该器件中，然后每个时间脉冲依次向左或右移动一个比特，在输出端进行输出。这种移位寄存器是一维的，事实上还有多维的移位寄存器，即输入、输出的数据本身就是一些列位。实现这种多维移位寄存器的方法可以是将几个具有相同位数的移位寄存器并联起来。

本项目使用的74HC595为一种并行输入转串行输出的移位寄存器

简单概括工作原理为：

8个按键为一组，共同接在74HC595移位寄存器上

=========================================

——————————————————————————2.1

=========================================

= = = = = = = =

=========================================

1.1 1.2 1.3 1.4 1.5 1.6 1.7 1.8

当本组按键中有任意键被按下时，如 1.2按键被按下

这个74HC595会已一定速度循环依次发送

010000000

信号给单片机的信号引脚

（这里忽略了寻址，时钟信号等详细的解释，只是解释了单个并行输入转串行输出的简单原理，并且涉及到总线驱动知识，在后面的章节中会讲到）

当单片机接受到 2.1的 010000000信号时，就会判断出2.1下面所连接的1.2有电平变化。从而知道那个按键被按下

优缺陷点

相比于市面上的矩阵键盘方案来说，使用74HC595的优点有

  1. pcb元件少、布局简洁：不需要矩阵键盘每个键上都有的的二极管，并且布线少，每个按键只需要和对应的移位寄存器连接，不需要组成网络
  2. 算法简洁：只需要对移位寄存器输入判断，去抖动，而不用担心矩阵键盘的的鬼键，实现优雅的全键无冲

缺点：

  1. 移位寄存器相较于二极管，价格翻几十倍，不适合量产
